# QR Tool – Free QR Code Generator

Live site:
https://instantqr.io

A clean, browser-based QR Code Generator supporting:

- Text / URL QR codes
- Wi-Fi QR codes
- Adjustable size
- PNG download
- No data sent to any server

## Features

- Fully client-side
- No tracking
- Mobile friendly
- Free to use
