InstantQR — Privacy-First Online Tools
🌐 Live Site: [https://instantqr.io](https://instantqr.io)

InstantQR is a lightweight, browser-based utility platform focused on **privacy-first tools** that run entirely in your browser.

**No accounts. No tracking. No data stored.**

---

## Available Tools

### QR Code Generator

Create QR codes instantly for:

* Text / URL
* Wi-Fi networks (WPA / WEP / Open)
* Adjustable size (128–1024px)
* Error correction levels
* PNG download

All QR generation runs locally using client-side JavaScript.

### Password Generator

Generate secure passwords instantly with:

* Adjustable length (6–128)
* Lowercase, uppercase, numbers, symbols
* Ambiguous character filtering (O/0, l/1, I)
* Presets (PIN, Memorable, Max Strength)
* Entropy/strength estimation
* One-click copy

Passwords are generated locally using the Web Crypto API. Nothing is transmitted or stored.

## Privacy Principles

* 100% client-side execution
* No server-side processing
* No user data collection
* No password storage
* No QR payload storage
* Minimal analytics (Google Analytics for traffic insights only)

## Technical Stack

* Vanilla HTML / CSS / JavaScript
* QRious (QR generation)
* Web Crypto API (secure randomness)
* GitHub Pages hosting
