      What WHOIS shows
      
        
      
          WHOIS data helps you verify domain ownership signals and troubleshoot DNS issues.
          RDAP is the modern, structured way to access domain registration information.
        
      

        
      
          
      Common use cases
      
          
        
            
      • Verify registrar before transferring a domain.
        • 
            
      • Check expiration dates to prevent downtime.
        • 
            
      • Confirm nameservers when DNS changes aren’t propagating.
        • 
            
      • Investigate domain status codes (e.g., clientTransferProhibited).
        • 
          
      
        
      

        
      
          
      Domain status codes (examples)
      
          
        
            
      • clientTransferProhibited — registrar lock enabled.
        • 
            
      • clientUpdateProhibited — updates restricted.
        • 
            
      • pendingDelete — domain is pending deletion.
        • 
          
      
        
      

        
      
          
      Tips for better results
      
          
        
            
      • Enter only the domain (e.g., example.com), not a full URL.
        • 
            
      • If results look incomplete, the registry may redact fields for privacy.
        • 
            
      • Use DNS Lookup to verify records after updates.
        •
