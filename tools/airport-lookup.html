What this tool does
Airport Lookup helps travelers quickly identify an airport by code, city, or name. It is useful when checking flight details, planning routes, comparing airports, or confirming the correct airport code before booking.
IATA vs ICAO
IATA codes are the short three-letter codes most travelers recognize. ICAO codes are four-letter aviation identifiers often used in operational and technical contexts.

FAQ

What is an IATA code?

An IATA code is a three-letter airport code used by airlines and travelers, such as PHX, LAX, or JFK.

What is an ICAO code?

An ICAO code is a four-letter airport identifier often used in aviation operations and flight planning.

Can I search by city or airport name?

Yes. You can search by city, airport name, IATA code, or ICAO code.

Does the map update when I select an airport?

Yes. Click Show on Map for any result and the map will center on that airport.
