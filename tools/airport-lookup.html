FAQ

What is an IATA code?

An IATA code is a three-letter airport code used by airlines and travelers, such as PHX or LAX.

What is an ICAO code?

An ICAO code is a four-letter airport identifier often used in aviation operations and flight planning.

Can I search by city or airport name?

Yes. You can search by city, airport name, IATA code, or ICAO code.

Does this tool show live flight status?

No. This version focuses on airport directory details such as codes, location, timezone, and coordinates.
