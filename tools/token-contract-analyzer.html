FAQ

What does this token contract analyzer show?

This tool checks bytecode presence, token metadata, transfer behavior, proxy slots, recent logs, and basic suspicious signals across supported EVM chains.

Does this tool perform a full smart contract audit?

No. This tool is a fast first-pass analyzer and does not replace professional auditing or source code review.

Which networks are supported?

The analyzer supports Ethereum, Base, Arbitrum, Optimism, BNB Smart Chain, Polygon, and Avalanche C-Chain.
