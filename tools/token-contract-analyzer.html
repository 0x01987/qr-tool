Interpretation
—
This tool is a quick on-chain metadata checker. It does not verify ownership controls, hidden logic, taxes, upgradeability, blacklist behavior, or full audit safety.

Useful for quick screening

This is helpful when you want to confirm that an address is actually a deployed contract and whether it responds like a standard ERC-20 token.

Not a full security audit

A contract can return standard metadata and still include risky or malicious logic. Use this as a first-pass inspection tool, not a final trust decision.

FAQ

What does this token contract analyzer show?

This tool shows whether bytecode exists at an address, basic token metadata like name and symbol, decimals, total supply, and whether common ERC-20 functions respond.

Does this tool audit contract safety?

No. This tool is a lightweight on-chain contract inspector and does not replace a professional smart contract audit.

Does this tool run in the browser?

Yes. The analysis runs from your browser using public Ethereum RPC endpoints.
