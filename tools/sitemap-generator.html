FAQ

Why use a separate sitemap-pages.txt file?

It gives InstantQR one clean source of truth. Add or remove a page once, then regenerate sitemap.xml anytime.

Do I need a server for this?

No. This tool runs in the browser and simply reads your text file, builds XML, and lets you download the result.

What if the page list file changes?

Click Reload Page List, then Generate Sitemap again.
