What this tool does

It decodes Base64URL JWT segments and renders the header and payload as readable JSON for fast inspection.

What it does not do

It does not verify the token signature or confirm whether the token is trusted or authentic.

FAQ

Does this JWT decoder verify the signature?

No. This tool decodes and displays token parts, but it does not verify the JWT signature.

Does this tool upload my token?

No. JWT decoding runs locally in your browser.

Can I view exp and iat timestamps?

Yes. This tool attempts to render common Unix timestamp claims such as exp, iat, and nbf into readable dates.
