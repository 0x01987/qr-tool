What this tool measures

This tool sends repeated browser requests to a target website and measures roughly how long each response takes. It is useful for checking website responsiveness, rough latency, and basic reachability from your browser.

Latency guide

  • Under 20 ms — Excellent
  • 20–50 ms — Very good
  • 50–100 ms — Acceptable
  • 100 ms+ — Slower response

Why tests can fail

Some websites block browser requests, require stricter permissions, or handle redirects differently. A failed test does not always mean the server is down.
