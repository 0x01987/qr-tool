    What WHOIS tells you
    
        
    
          WHOIS data helps with domain transfers, DNS troubleshooting, and basic security checks.
          This page uses RDAP (structured WHOIS) and selects the correct RDAP server using the IANA RDAP bootstrap.
        
    

        
    
          
    Common use cases
    
          
      
            
    • Check expiration dates to prevent downtime.
      • 
            
    • Confirm nameservers after DNS updates.
      • 
            
    • See status codes like clientTransferProhibited (domain lock).
      • 
            
    • Verify registrar before initiating a transfer.
      • 
          
    
        
    

        
    
          
    Status codes (examples)
    
          
      
            
    • clientTransferProhibited — transfer lock enabled
      • 
            
    • clientUpdateProhibited — updates restricted
      • 
            
    • pendingDelete — domain pending deletion
      • 
          
    
        
    

        
    
          Note: Some registry RDAP servers may block browser requests (CORS). When that happens, this tool automatically retries via a fallback fetch method to keep it working on static hosting.
