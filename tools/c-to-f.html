Quick notes

Temperature conversion is not a simple constant ratio. Celsius and Fahrenheit use different zero points, so an offset is part of the formula.

Formula
°F = (°C × 9/5) + 32
Examples
0 °C = 32 °F
25 °C = 77 °F
100 °C = 212 °F

How this converter works

This tool converts Celsius to Fahrenheit instantly and also supports reverse conversion if you type into the Fahrenheit field. It is designed to be quick, responsive, and easy to use on both desktop and mobile.

Celsius and Fahrenheit measure the same physical quantity, but they use different scales. That is why the conversion formula includes both multiplication and an added offset.
