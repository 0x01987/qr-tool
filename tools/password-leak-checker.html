Password Leak Checker

Check if your password appears in known breach datasets using privacy-first k-anonymity. Your full password is never sent.

Privacy mode keeps the check offline and only gives local strength guidance. Turn it off to run the k-Anonymity breach lookup, where only the first 5 characters of the SHA-1 hash prefix are sent.

How privacy works

  • Password is hashed locally in your browser.
  • Only the first 5 hash characters are sent.
  • Your browser compares results locally.

If it is found

Change it everywhere you used it, switch to a unique password, and enable MFA. Use the Password Generator for a stronger replacement.
