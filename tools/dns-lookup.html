FAQ


          

            What is a DNS lookup?
            
A DNS lookup checks the DNS records for a domain or hostname, such as A, AAAA, CNAME, MX, TXT, NS, and SOA records.

          


          

            What DNS record types can I check here?
            
This DNS lookup tool supports A, AAAA, CNAME, MX, TXT, NS, SOA, and an ANY-style multi-record lookup.

          


          

            Does this tool use DNS-over-HTTPS?
            
Yes. Your query is sent directly from your browser to the selected DNS-over-HTTPS resolver, such as Cloudflare or Google.

          


          

            Why would a DNS lookup return no answers?
            
A lookup can return no answers if the domain has no records for that type, the hostname is incorrect, the resolver does not return results for that query, or the domain has not been configured yet.
