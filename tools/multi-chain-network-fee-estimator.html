FAQ

Which networks does this fee estimator support?

This tool supports Bitcoin, Litecoin, Dogecoin, Dash, and TRON. Each network uses a different fee model, so the estimator adapts to that chain.

Why are these network fees estimated differently?

Bitcoin usually uses sat/vB. Litecoin, Dogecoin, and Dash typically use per-kilobyte fee estimates. TRON uses bandwidth and energy instead of the same UTXO fee model.

Is this the exact final fee?

No. It is a practical estimate based on current data and your chosen inputs. Final wallet fees can vary depending on the wallet, transaction type, and network conditions.

Do you store my endpoint URL or input?

No. This page runs in your browser and sends requests directly to the endpoint you choose.

Why can a public endpoint fail?

Public endpoints may be rate-limited, blocked by CORS, overloaded, or temporarily unavailable. If that happens, try again later or switch to another endpoint.
