What is a Port Checker?

A port checker allows you to test whether a specific port on a server is reachable from the internet. This is useful for troubleshooting firewalls, server configurations, and network connectivity.

Common Ports

  • 80 – HTTP
  • 443 – HTTPS
  • 22 – SSH
  • 21 – FTP
  • 25 – SMTP
  • 3306 – MySQL
