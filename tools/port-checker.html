FAQ

Can browsers scan any TCP port?

No. Browsers cannot perform raw TCP scans. This tool checks if a browser can reach a web service on a specific port.

What ports work best?

Ports used by web services such as 80, 443, 8080, and 8443.

Why might a port appear closed?

The server may block browser requests, require authentication, or use a protocol not supported by browsers.
