What reverse DNS does

Reverse DNS resolves an IP address back to a hostname using a PTR record. It is commonly used for email server validation, logging, and network analysis.

Why no PTR record may appear

Many IP addresses do not have reverse DNS configured. If no PTR record is set by the IP owner or provider, the lookup will return no result.

FAQ

What is reverse DNS lookup?

Reverse DNS lookup resolves an IP address to a hostname using a PTR record. It is the opposite of a normal DNS lookup.

What is a PTR record?

A PTR record is a DNS record that maps an IP address back to a hostname.

Why does an IP have no PTR record?

Not every IP address has reverse DNS configured. The network owner or hosting provider must create the PTR record.

Does this tool support IPv6 reverse DNS?

Yes. This reverse DNS lookup tool supports both IPv4 and IPv6 addresses.
