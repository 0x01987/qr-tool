How it works

  1. Enter your Wi-Fi name (SSID) and password.
  2. Select security type (WPA/WEP/No password) and whether the network is hidden.
  3. Click Generate and download your QR code as a PNG.

Tip: test-scan before printing and placing it near your router or front desk.

QR payload format: WIFI:T:WPA;S:SSID;P:password;H:false;;
Your WiFi QR code will appear here.

What is a WiFi QR code?

A WiFi QR code lets guests connect to your wireless network by scanning a code with their phone camera. It contains your SSID, password, and security type in a standard format supported by many iPhone and Android devices.

Best practices

  • Use WPA/WPA2/WPA3 when possible.
  • Double-check spelling because SSID is case-sensitive.
  • Print a copy and keep a backup PNG saved.
  • Use the hidden network option only if your router actually uses a hidden SSID.
