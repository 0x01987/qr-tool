NFT Rarity Checker

Upload NFT metadata JSON to analyze trait rarity and rarity scores.

What is NFT Rarity?

NFT rarity measures how uncommon certain traits are within a collection. NFTs with rare attributes tend to have higher value in many collections.

How Rarity is Calculated

Trait rarity is calculated by dividing the total collection size by the number of NFTs containing a specific trait.
