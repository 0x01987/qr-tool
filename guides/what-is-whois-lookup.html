What Is WHOIS Lookup?

WHOIS lookup helps you understand who manages a domain and how it’s configured. You’ll commonly see the registrar, key dates, status codes, and nameservers. Today, many registries provide this data via RDAP (a modern replacement for classic WHOIS).

What WHOIS shows

A WHOIS lookup is like a public “registration record” for a domain. Depending on the domain and registry privacy rules, you may see:

  • Registrar (where the domain is registered)
  • Registration date (when it was created)
  • Expiration date (when it needs renewal)
  • Domain status codes (locks/restrictions)
  • Nameservers (where DNS is hosted)
  • Contact data (often redacted)

On many modern domains, personal owner details are hidden, but technical details still help you troubleshoot and verify legitimacy.

WHOIS vs RDAP

Classic WHOIS was a text-based protocol. RDAP (Registration Data Access Protocol) is a modern API that returns structured JSON. That structure makes it easier for tools to display consistent results and parse fields like registrar, events, and nameservers.

InstantQR uses RDAP where available because it’s cleaner, more consistent, and easier to render quickly in the browser.

Key fields to look for

  • Registrar: confirms who manages the domain registration.
  • Expires: if a domain is near expiration, it may be at risk of downtime or takeover.
  • Nameservers: tells you where DNS is hosted (often Cloudflare, Route 53, etc.).
  • Status: indicates locks (transfer/update restrictions) or lifecycle states like deletion.

Common use cases

  • Domain transfer prep: check the registrar and whether the domain is locked.
  • DNS troubleshooting: verify nameservers match where you expect DNS to be managed.
  • Security checks: examine suspicious domains (newly registered + odd nameservers can be a signal).
  • Ops hygiene: monitor expiration dates for business domains.

Common domain status codes

Status codes can look confusing, but a few are very common:

  • clientTransferProhibited — transfer lock enabled (often a good thing).
  • clientUpdateProhibited — updates restricted.
  • pendingDelete — domain is pending deletion (high risk).

FAQ

Can WHOIS show the domain owner?

Sometimes, but many registries redact personal information. You’ll usually still see registrar, dates, status, and nameservers.

What if WHOIS results look incomplete?

That’s often due to privacy redaction. Try validating DNS records too using DNS Lookup.

Is WHOIS lookup safe to use?

Yes—WHOIS/RDAP returns public registration data. Avoid pasting private credentials anywhere. InstantQR does not require accounts.
