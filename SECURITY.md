# Security Policy

## Supported Versions

This project is currently maintained on the `main` branch.

Security updates will be applied to the latest version only.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead:

- Email: 0x01987db@gmail.com  
OR  
- Open a private security advisory via GitHub

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will respond within 5 business days.

---

## Scope

This project is fully client-side and does not:

- Store user data
- Send QR data to any server
- Track user behavior

However, any vulnerabilities affecting user privacy, script injection, or dependency integrity should be reported.

Thank you for helping keep this project secure.
