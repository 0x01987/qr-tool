# Contributing to QR Tool

Thank you for your interest in contributing to this project!

This project aims to provide a clean, lightweight, and fully client-side QR code generator that is easy to use and privacy-friendly.

---

## How You Can Contribute

You can help improve this project by:

- Reporting bugs
- Suggesting features
- Improving documentation
- Optimizing performance
- Improving UI/UX
- Submitting code improvements

---

## Reporting Issues

If you find a bug:

1. Open a new Issue
2. Clearly describe:
   - What happened
   - What you expected to happen
   - Your browser and device
3. Include screenshots if possible

Please check existing issues before opening a new one.

---

## Submitting Changes

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Test thoroughly
5. Submit a Pull Request

Please:

- Keep changes focused and minimal
- Follow clean and readable code practices
- Avoid unnecessary dependencies
- Ensure the tool remains fully client-side

---

## Development Guidelines

- Keep the project lightweight
- No tracking scripts
- No unnecessary third-party libraries
- Prioritize performance and privacy

---

## Code Style

- Use clear, descriptive variable names
- Maintain consistent formatting
- Write clean, readable JavaScript
- Keep CSS modular and minimal

---

## Feature Requests

Feature ideas are welcome. Please open an Issue and describe:

- The problem the feature solves
- How it improves the tool
- A simple proposed implementation

---

## Questions

If you have general questions, open an Issue and we will respond as soon as possible.

---

Thank you for helping improve QR Tool!
